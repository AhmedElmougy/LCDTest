# LCDTest
 LCD test application for testing LCD user built driver for atmega32. 
 Driver is configurable you can choose :
 - 8bit or 4bit mode 
 - Font size
 - Number of lines 
 driver comes with pre configured custom characters that can be configured in HLCD_Config.h file
